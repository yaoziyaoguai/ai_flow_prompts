# No Env No Secrets —— 禁止读取敏感文件片段

## 适用场景

- 嵌入到任何编程类模板中作为安全约束
- 防止 Agent 读取 .env、密钥文件等敏感内容

## 完整片段

---

## 安全约束：敏感文件

以下文件类型**严格禁止读取**：

- `.env` / `.env.local` / `.env.production` 等环境变量文件
- `*.key` / `*.pem` / `*.p12` 等密钥文件
- `credentials*` / `secret*` / `token*` 等凭证文件
- 任何可能包含 API key、密码、token 的配置文件

如果任务需要这些文件中的信息：
- 要求用户提供具体的环境变量名
- 要求用户提供非敏感的值
- 不要尝试自行读取这些文件

---

## 使用提醒

- 这是一个**可嵌入片段**，不是独立模板
- 嵌入位置：通常和 `no-push-no-tag.md` 一起放在 prompt 末尾的安全约束部分
- 如果项目配置了 AGENTS.md 且已包含此规则，模板中可省略
